Project Overview

Dataset Description

This project analyses workout information collected by an individual's smartwatch over the course of a month. The dataset, whilst relatively small, presented several data quality challenges that provided excellent practice for data cleaning techniques using Python's Pandas library.

Objective

To demonstrate data cleaning and preparation skills by handling missing values, inconsistencies, and data quality issues whilst minimising data loss. The cleaned dataset was then visualised using Matplotlib to reveal workout patterns over time.

Raw data (left) vs cleaned data (right)

Data Cleaning Process

1. Initial Mean Imputation Challenge

Initially, I filled gaps in the Calories Burned column using the overall mean. However, this approach skewed the data significantly. For example, a 45-minute workout would appear to burn significantly more calories than similar entries.

Initial mean imputation showing data skewing

2. Context-Aware Mean Calculation

To address the skewing issue, I calculated separate means based on workout duration, allowing for more context-aware imputation:

  • 45-minute workouts: 285 calories (mean)
  • 60-minute workouts: 314 calories (mean)

Missing values were excluded from mean calculations to ensure accuracy.

Context-aware mean calculation by workout duration

3. Appropriate Mean Imputation

Applied the duration-specific means to replace missing calorie values, ensuring each imputation was contextually appropriate for the workout length.

Replacing missing values with duration-specific means

4. Date Formatting and Data Consistency

Addressed missing date values and corrected data inconsistencies. Since the dataset was small, I prioritised minimising data loss by correcting rather than deleting problematic entries.

Inserting missing dates and formatting corrections

5. Final Data Corrections

Corrected obvious data entry errors (e.g., 450 calories likely meant 45 minutes) and rounded calorie values to integers for consistency.

Correcting suspected data entry errors

Key Insights

Workout Intensity Decline

The analysis revealed a drop in workout intensity over the 30-day period, suggesting potential motivation or fatigue patterns.

Data Quality Challenges

Small datasets require careful handling of missing values to avoid significant data loss whilst maintaining analytical integrity.

Context-Aware Imputation

Duration-specific mean imputation proved more accurate than overall mean imputation for calorie data.

Project Reflections

Technical Skills Developed

  • Pandas Data Manipulation: Gained experience with data cleaning, missing value imputation, and data type conversions
  • Problem-Solving: Developed context-aware solutions for data quality issues
  • Matplotlib Visualisation: Created clean, informative plots despite dataset limitations
  • Data Quality Assessment: Learned to identify and address various data inconsistencies

Key Learnings

This project reinforced the importance of understanding your data before applying cleaning techniques. The initial mean imputation approach highlighted how seemingly logical solutions can introduce bias. The experience of working with a small, imperfect dataset provided valuable practice in making data-driven decisions about cleaning strategies.

Future Applications

The techniques learned in this project - particularly context-aware imputation and careful handling of small datasets - are directly applicable to real-world data analysis scenarios where data quality issues are common.
